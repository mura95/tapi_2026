# ReminderNotificationUI デザイン仕様書

## 概要

高齢者向けリマインダー通知画面のUIデザイン。
服薬・食事・水分補給などの時間になると、犬が吠えて通知し、この画面が表示される。

**ターゲットユーザー:** 高齢者（65歳以上を想定）
**デバイス:** Android タブレット（10インチ程度）
**表示タイミング:** リマインダー時間になったとき（犬が吠えている状態）

---

## 必要な要素

### 1. 背景オーバーレイ

| 項目 | 仕様 |
|------|------|
| **目的** | 背景（犬）を暗くし、通知に集中させる |
| **範囲** | 画面全体 |
| **色** | 黒 50〜70% 透過 |
| **タップ動作** | タップで「完了」と同じ動作（任意） |

---

### 2. 通知パネル（メインカード）

| 項目 | 仕様 |
|------|------|
| **目的** | 通知内容を表示するメインエリア |
| **位置** | 画面中央 |
| **サイズ** | 横: 画面幅の50〜60%、縦: コンテンツに応じて |
| **背景色** | 白または明るい色 |
| **角丸** | 16〜24px 推奨 |
| **影** | ドロップシャドウ推奨（浮いている感を出す） |
| **余白** | 内側 32〜48px |

---

### 3. アイコン

| 項目 | 仕様 |
|------|------|
| **目的** | 通知の種類を視覚的に伝える |
| **サイズ** | 64×64px 〜 96×96px |
| **位置** | パネル上部中央 |
| **種類** | 6種類（下記参照） |

#### アイコン種類

| タイプ | 日本語 | アイコンイメージ | 推奨色 |
|--------|--------|-----------------|--------|
| medication | 服薬 | 薬・カプセル・錠剤 | 青・緑 |
| meal | 食事 | お椀・箸・食器 | オレンジ・茶 |
| hydration | 水分補給 | コップ・水滴 | 水色・青 |
| exercise | 運動 | 人・体操・歩く人 | 緑・黄緑 |
| rest | 休憩 | 椅子・ソファ・Zzz | 紫・青 |
| appointment | 予定 | カレンダー・時計 | 赤・オレンジ |

**備考:** シンプルで認識しやすいフラットデザイン推奨

---

### 4. タイプラベル

| 項目 | 仕様 |
|------|------|
| **目的** | 通知の種類をテキストで明示 |
| **位置** | アイコンの下 |
| **フォントサイズ** | 24〜32px |
| **フォントウェイト** | Medium または Bold |
| **色** | グレー or タイプに応じた色 |

#### 表示テキスト

| タイプ | 表示テキスト |
|--------|-------------|
| medication | 服薬 |
| meal | 食事 |
| hydration | 水分補給 |
| exercise | 運動 |
| rest | 休憩 |
| appointment | 予定 |

---

### 5. メインメッセージ

| 項目 | 仕様 |
|------|------|
| **目的** | 何をすべきか明確に伝える |
| **位置** | タイプラベルの下、パネル中央 |
| **フォントサイズ** | **48〜64px（大きく）** |
| **フォントウェイト** | Bold |
| **色** | 黒または濃いグレー |
| **行揃え** | 中央揃え |
| **行間** | 1.4〜1.6 |

#### メッセージ例

| タイプ | メッセージ |
|--------|-----------|
| medication | ○○さん、お薬を服用してください |
| meal | ○○さん、お食事の時間です |
| hydration | ○○さん、お水を飲んでください |
| exercise | ○○さん、体操の時間です |
| rest | ○○さん、休憩してください |
| appointment | ○○さん、予定があります |

**備考:** 「○○さん」はユーザー設定の名前。未設定の場合は省略。

---

### 6. 完了ボタン

| 項目 | 仕様 |
|------|------|
| **目的** | 通知を確認したことを伝えるアクション |
| **位置** | パネル下部中央 |
| **サイズ** | 横: 200〜300px、縦: 64〜80px |
| **テキスト** | 「完了」 |
| **フォントサイズ** | 32〜40px |
| **フォントウェイト** | Bold |
| **背景色** | プライマリカラー（青・緑など） |
| **テキスト色** | 白 |
| **角丸** | 8〜16px |
| **状態** | 通常 / 押下時（色が少し暗くなる） |

**重要:** 高齢者が押しやすいよう、十分な大きさを確保

## デザイン要件

### アクセシビリティ

| 項目 | 要件 |
|------|------|
| **コントラスト比** | 4.5:1 以上（WCAG AA準拠） |
| **タップ領域** | 最小 48×48px（完了ボタンはそれ以上） |
| **フォント** | 読みやすいゴシック体（游ゴシック、Noto Sans JP等） |
| **色覚異常対応** | 色だけでなくアイコン・テキストでも区別可能に |

### 視認性

- **高コントラスト:** 背景と文字のコントラストを十分に
- **大きな文字:** メインメッセージは最低48px
- **シンプル:** 情報を詰め込みすぎない
- **余白:** 要素間に十分なスペース

## 状態バリエーション

### 通常状態
- 標準的な表示

### ボタン押下状態
- 完了ボタンの色が少し暗くなる
- または軽いアニメーション（スケール0.95など）

## 実装上の注意（エンジニア向け）

### Unity での設定

- **Canvas:** Screen Space - Overlay
- **Sort Order:** 1000（最前面）
- **Canvas Scaler:** Scale With Screen Size
- **Reference Resolution:** 1920×1080

### コンポーネント参照

```
ReminderNotificationUI (Canvas)
├── Background (Image + Button) → _backgroundButton
└── Panel (Image)
    ├── IconImage (Image) → _iconImage
    ├── TypeText (TMP) → _typeText
    ├── MessageText (TMP) → _messageText
    └── CompleteButton (Button) → _completeButton
```
